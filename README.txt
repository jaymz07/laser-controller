Program for generating x/y parametric signals from an on-screen drawing.
Uses the sound card to generate analog voltages suitable (after amplification) for driving a laser projector.

--------------------------
Usage
--------------------------
To run program:
java GraphicalLaserOutput

  ----------------
  User inputs:
    
    Mouse:
      Click or drag to add points to laser path.
    
    Keyboard:
      Backspace --	Remove last point drawn
      C		--	Clear entire drawing
